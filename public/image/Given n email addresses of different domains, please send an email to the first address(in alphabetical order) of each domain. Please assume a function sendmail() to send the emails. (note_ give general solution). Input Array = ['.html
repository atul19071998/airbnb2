      Question
      Given n email addresses of different domains, please send an email to the first address(in alphabetical order) of each domain
      Answer · 0 votes
      You could reverse-sort the list and use the domain as a key to a dict. When you insert into the dict, the last remaining value would be the first email from that domain in alphabetical order, running in O(n) time. a = ['ghi@hotmail.com', 'def@yahoo.com', 'ghi@gmail.com', 'abc@channelier.com', 'abc@hotmail.com', 'def@hotmail.com', 'abc@gmail.com', 'abc@yahoo.com', 'def@channelier.com','jkl@hotmail.com', 'ghi@yahoo.com', 'def@gmail.com'] b = {} for i in sorted(a)[::-1]: # Note this only works because emails can't have more than on '@' symbol b[i.split("@")[1]] = i print(b.values()) # dict_values( # [ # "abc@hotmail.com", # "abc@yahoo.com", # "abc@gmail.com", # "abc@channelier.com", # ] # )
